
## Tic Tac Toe implementation
This is an implementation of tic-tac-toe where you can choose the players. It can be.

- Human vs Human
- Human vs RandomAI
- Human vs SmartAI
- RandomAI vs RandomAI
- RandomAI vs SmartAI
- SmartAI vs SmartAI
- And the of the pairings above.
